# AWS Cloud Resume Challenge

Designed and deployed a fully serverless cloud resume website hosted on AWS with automated CI/CD pipelines and Infrastructure as Code.

## Architecture

Frontend:
- Hugo static site generator
- Hosted on Amazon S3
- Delivered globally via CloudFront CDN
- Custom domain configuration

Backend:
- API Gateway
- AWS Lambda
- DynamoDB visitor counter

CI/CD:
- GitHub Actions deployment pipelines
- Automated frontend deployment to S3
- Automated backend deployment using AWS SAM

Infrastructure as Code:
- AWS SAM templates
- Serverless infrastructure provisioning
- Repeatable backend deployments

# Engineering Challenges Solved

## Deployment Rollbacks

Diagnosed and resolved deployment failures during CI/CD pipeline updates and backend infrastructure changes.

## Environment Mismatch Issues

Resolved inconsistencies between local development and production deployment environments, particularly during Hugo static site builds and AWS deployments.

## Artifact Packaging Problems

Learned how deployment artifacts are generated, packaged and uploaded during serverless deployments using AWS SAM and GitHub Actions workflows.

## Infrastructure Drift Awareness

Improved understanding of infrastructure drift and the importance of Infrastructure as Code for maintaining consistent cloud environments.

## CI/CD Troubleshooting

Debugged GitHub Actions workflow failures involving:
- Hugo builds
- AWS credential configuration
- S3 deployment sync
- CloudFront invalidation
- Theme submodule loading
